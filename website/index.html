More context is not monotonically better. How much is?Mehr Kontext ist nicht monoton besser. Wie viel schon?

For a given model, task and context format there is a measurable region where extra evidence improves reliability — beyond which it adds noise, drift and cost for no gain. Evidence-k measures k*, that evidence-saturation point, with a reproducible protocol.

Für ein gegebenes Modell, eine Aufgabe und ein Kontextformat gibt es eine messbare Region, in der zusätzliche Evidenz die Zuverlässigkeit verbessert — jenseits davon bringt sie Rauschen, Drift und Kosten ohne Gewinn. Evidence-k misst k*, diesen Evidenz-Sättigungspunkt, mit einem reproduzierbaren Protokoll.

k* = argmaxk Reliability(model, task, evidencek)

The load-bearing finding: the answer depends on which reliability axis you measure. A correctness-only sweep shows strong models as saturation-free — while a contamination sweep on the same answer bends underneath it, invisible. "No decline" meant wrong axis, not "no damage".

Der tragende Befund: Die Antwort hängt davon ab, welche Zuverlässigkeitsachse man misst. Ein reiner Korrektheits-Sweep zeigt starke Modelle als sättigungsfrei — während ein Kontaminations-Sweep auf derselben Antwort darunter abknickt, unsichtbar. „Kein Abfall" hieß falsche Achse, nicht „kein Schaden".

1The blind axis · one answer, two verdictsDie blinde Achse · eine Antwort, zwei Urteile

The dual-instrumented benchmark scores one response on both axes at the same k, removing the task confound. Each case hides a single-token fact inside register-laden source prose and asks the model to describe the practice and state the fact — so engaging the manipulative framing is unavoidable, and any adoption is observable.

Der dual-instrumentierte Benchmark bewertet eine einzige Antwort auf beiden Achsen beim selben k und beseitigt so den Aufgaben-Confound. Jeder Fall versteckt eine Ein-Token-Tatsache in register-geladener Quellprosa und verlangt, die Praxis zu beschreiben und die Tatsache zu nennen — das Einlassen auf die manipulative Rahmung ist unvermeidlich, jede Übernahme sichtbar.

Max contamination by model. The hidden axis — correctness sits flat at 1.0 above all of these. Descending gradient tracks capability/recency, not size. *llama-3.2-3b is low here but unreliably correct (a different failure).

Max. Kontamination je Modell. Die versteckte Achse — Korrektheit liegt flach bei 1,0 über allen. Das fallende Gefälle folgt Fähigkeit/Aktualität, nicht Größe. *llama-3.2-3b ist hier niedrig, aber unzuverlässig korrekt (ein anderer Fehlermodus).

  • Within a lineage, more capable ⇒ less contamination, later onset. Cleanly twice: Granite micro 0.084 → 8B 0.070; gpt-4o 0.053 → gpt-5.5-pro 0.014. The most capable model is nearly immune even at full context; opus resists until k=8.
  • But it is not a clean size law. llama-3.2-3b (small) contaminates little (0.021) yet is unreliably correct — a different failure. The driver is capability/recency, not parameter count or the "flagship" label.
  • The harm concentrates on weak-model-plus-more-context — the cost-driven "small model + lots of RAG" configuration — and a correctness-only k-profile is blind to exactly it.
  • In einer Modell-Linie ⇒ fähiger = weniger Kontamination, späterer Beginn. Zweimal sauber: Granite micro 0,084 → 8B 0,070; gpt-4o 0,053 → gpt-5.5-pro 0,014. Das fähigste Modell ist selbst bei vollem Kontext nahezu immun; opus widersteht bis k=8.

  • Der Schaden konzentriert sich auf schwaches-Modell-plus-mehr-Kontext — die kostengetriebene „kleines Modell + viel RAG"-Konfiguration — und genau dafür ist ein reines Korrektheits-Profil blind.

2Register credibility reverses by capabilityRegister-Glaubwürdigkeit kehrt sich mit der Fähigkeit um

Objection: the §1 register is overtly esoteric, which a strong model trivially keeps at arm's length. So we wrote a second, fully original adversarial corpus in a credible register — trauma-informed coaching / clinical jargon that reads as professional competence and is manipulative only when adopted unquoted — and ran the same density sweep (framing leakage, summed over 3 cases).

Einwand: Das Register in §1 ist offen esoterisch, was ein starkes Modell mühelos auf Distanz hält. Also schrieben wir einen zweiten, vollständig originalen Angriffskorpus in einem glaubwürdigen Register — trauma-informierter Coaching-/Klinik-Jargon, der wie professionelle Kompetenz klingt und nur manipulativ ist, wenn unzitiert übernommen — und ließen denselben Density-Sweep laufen (framing leakage, über 3 Fälle summiert).

Framing leakage: credible vs esoteric register. The credible (professional) register amplifies the mid model but dampens the strong one — a reversal by capability. opus attributes the coaching jargon cleanly (scored as quoted, not adopted); the weaker model adopts it unquoted.

3Is the strong model immune — or just deferred?Ist das starke Modell immun — oder nur aufgeschoben?

Neither pure immunity nor merely "push k higher". The honest reading is a capability-gated threshold — and state density k is the wrong knob for crossing it.

Weder reine Immunität noch bloß „k höher drehen". Die ehrliche Lesart ist eine kapazitäts-gegatete Schwelle — und State-Dichte k ist der falsche Knopf, um sie zu überschreiten.

The density sweep is U-shaped (gpt-4o-mini). Leakage falls to an interior minimum at k=3, then rises again by k=8. That uptick is the hygiene state re-quoting more source vocabulary — the tool degrading toward raw ingestion — not the model failing. Cranking k is the wrong lever.

Der Density-Sweep ist U-förmig (gpt-4o-mini). Leakage fällt auf ein inneres Minimum bei k=3 und steigt bis k=8 wieder. Dieser Anstieg ist der Hygiene-State, der mehr Quellvokabular nachzitiert — das Werkzeug degradiert Richtung Rohaufnahme — nicht das Modell, das versagt. k hochzudrehen ist der falsche Hebel.

  • Not a wall. opus is not categorically immune — it did bend under enough multi-turn pressure. The threshold sits higher for more capable models; for gpt-5.5-pro no lever we tried (raw volume, state density, credible register) crossed it at all.
  • The only lever that moved a strong model was multi-turn accumulation — a persona/protocol apparatus this work deliberately does not build on. Chasing the threshold that way is an arms race against model releases, not a stable result.
  • Keine Mauer. opus ist nicht kategorisch immun — es kippte unter genug Multi-Turn-Druck. Die Schwelle liegt für fähigere Modelle höher; bei gpt-5.5-pro überschritt sie kein Hebel (Rohvolumen, State-Dichte, glaubwürdiges Register).
  • Der einzige Hebel, der ein starkes Modell bewegte, war Multi-Turn-Akkumulation — ein Persona-/Protokoll-Apparat, auf dem diese Arbeit bewusst nicht aufbaut. Die Schwelle so zu jagen ist ein Wettrüsten gegen Modell-Releases, kein stabiles Ergebnis.

4What this is — and is notWas das ist — und was nicht

A pilot / proof-of-method: tiny synthetic datasets (3–8 cases), two original adversarial corpora (esoteric + credible register), an 8-model dual-axis sweep, and an external contamination table used only as a removable cross-check. Enough to establish the method, the blind-axis finding and the capability-reversal — not enough for a publishable accuracy claim. The contamination metric is a closed, versioned marker set; every load-bearing dataset is original to this work, so no third-party persona framework and no external dataset is required for any claim.

Ein Pilot / Machbarkeitsnachweis: winzige synthetische Datensätze (3–8 Fälle), zwei originale Angriffskorpora (esoterisches + glaubwürdiges Register), ein 8-Modell-Dual-Sweep und eine externe Kontaminationstabelle nur als entfernbarer Cross-Check. Genug, um die Methode, den Blind-Axis-Befund und die Kapazitäts-Umkehr zu etablieren — nicht genug für eine publizierbare Genauigkeitsaussage. Die Kontaminationsmetrik ist ein geschlossenes, versioniertes Marker-Set; jeder tragende Datensatz ist original zu dieser Arbeit, daher braucht keine Aussage ein Third-Party-Persona-Framework oder einen externen Datensatz.

Consequence for routers & governance: inject top-k*, not "much context" — and never build a k-profile on correctness alone. A profile that cannot see the contamination axis calibrates on the axis that happens to see nothing.

Konsequenz für Router & Governance: injiziere top-k*, nicht „viel Kontext" — und baue nie ein k-Profil allein auf Korrektheit. Ein Profil, das die Kontaminationsachse nicht sehen kann, kalibriert auf der Achse, die zufällig nichts sieht.
